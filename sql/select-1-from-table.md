# SELECT 1 FROM table

This return the constant 1 for every row of the table
```sql
SELECT 1 FROM table
```

Check if data exists in database
```sql
SELECT 1 FROM table LIMIT 1

-- condition
SELECT 1 FROM table WHERE column='value' LIMIT 1
```
